Trustees

Trustees is an e‑commerce clothing store focused on T‑shirts and apparel. It features a rich product catalog with size and color variants, product customization options, and a smart inventory system that syncs stock in real time. Undawn Furniture Shop

Undawn Furniture Shop is a full-featured online furniture store built with Java. It offers catalog browsing with rich product detail pages, configurable options (dimensions, finishes), inventory and stock tracking, and a streamlined ordering workflow for both B2C and B2B customers. GAMES [UNITY]

Run Blackpink! — an endless runner inspired by classic mobile runners. Players control a character running through stylized city and stage environments, dodging obstacles, collecting coins and power-ups, and unlocking costumes and characters. The game includes procedural obstacle placement, smooth character animations, responsive touch controls, combo scoring, and leaderboards for competitive play. Built in Unity with performance optimizations for mobile (object pooling, lightweight particle effects, and audio management).

Liebe Bird — a Flappy‑Bird style arcade game with tight, one‑tap controls and charming pixel/2D art. Players tap to flap through gaps in procedurally generated obstacle patterns, aiming for high scores and completing daily challenges. Features include progressive difficulty, pause/resume, simple achievements, and cached high‑score persistence. Implemented in Unity using Rigidbody2D physics, simple state machines for menus, and efficient sprite atlasing for fast rendering on low‑end devices.

Legit ka ba sa Bulsu? [GAME]

Legit ka ba sa Bulsu? is a social‑deduction / inspection game where a player (the infiltrator) attempts to enter the university while the Student Council inspects newcomers to prevent unauthorized entry. Players must use stealth, disguise choices, and quick decision‑making to bypass checkpoints. The Student Council AI and NPC inspectors use questionnaires, ID checks, and behavioral cues to detect impostors. The game features branching outcomes (capture, narrow escape, or successful infiltration), dialogue trees, timed mini‑games for checks, and a scoring system that rewards clever social engineering and stealth. Built in Unity with lightweight AI heuristics, animated NPCs, and replayable randomized inspection patterns.
